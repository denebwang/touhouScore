# 星莲船相关开发笔记
## 地址
### 飞碟相关
碟碎: 0 = 无, 1 = 红, 2 = 蓝, 3 = 绿

0x004B0C4C 第一个碟碎 

0x004B0C50 第二个碟碎 

0x004B0C54 第三个碟碎 

0x004B0C58 碟碎数量

0x004B4534 飞碟指针

0x004B4534 + 0x48 红点数

0x004B4534 + 0x4C 蓝点数
### 分数相关
0x004B0C78 最大得点

0x004B0C44 分数

0x004B0CDC 擦弹
### 其他
0x004B0C90 人物(0 = 灵梦,1 = 马铃薯,2 = 早苗)

0x004B0C94 机体(0 = A机, 1 = B机)

0x004B0CB0 面数

0x004AEBD0 难度
